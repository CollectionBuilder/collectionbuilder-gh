{% comment %} Item figure from objectid. This include adds a figure to the page styled using bootstrap, https://getbootstrap.com/docs/4.5/content/figures/ It requires an "objectid" with the include, which is used to find the object title and image (for image or youtube objects). If multiple objectids are included (separated by ; ), they will be added to a "col-md" in a row which will automatically divided equally. E.G. --> {% include feature/item-figure.html objectid="demo_001" %} Options: - "objectid" = one or more objectids from this collection, separated by semicolon ; - "width" = will use responsive sizing to set the % size on desktop (will be 100% on mobile), choose from "25", "50", "75", or "100" - "caption" = by default the figure include automatically adds the title of the item from your metadata. The caption option allows you to manually add a different caption, or give the value false for none. - "float" = will use responsive float utility to add float on desktop (will not float on mobile), choose from "left" or "right" {%- endcomment -%} {%- assign figures = include.objectid | split: ";" %}
{% for i in figures %} {%- assign figure = site.data[site.metadata] | where: "objectid", i | first -%}
{% endfor %}
